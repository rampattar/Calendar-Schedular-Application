# Calendar Scheduler Application

**Calendar Scheduler Application** is a MERN stack project that allows users to manage events efficiently with features like login/signup, recurring tasks, reminders, and shared scheduling. The application supports real-time synchronization and secure user authentication.

---

## Features

- **User Authentication:** Secure login/signup using JWT.
- **Event Management:** Create, edit, and delete events.
- **Recurring Tasks:** Schedule tasks to repeat daily, weekly, or monthly.
- **Reminders:** Get notifications for upcoming events.
- **Shared Scheduling:** Collaborate and share events with others.
- **Real-time Sync:** Updates are reflected immediately for all users.
- **Responsive UI:** Fully responsive calendar interface across devices.

---

## Technologies Used

- **Frontend:** React.js, HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT (JSON Web Tokens)
- **Real-time Updates:** Socket.io (if implemented)
- **Version Control:** Git & GitHub
